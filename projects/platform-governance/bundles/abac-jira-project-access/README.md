# ABAC Jira Project Access Bundle

This bundle is the first ABAC dogfood boundary for Jira project-key row access.

It owns the focused `SPEC.md` for the Jira project access-map contract, policy-supporting UDF contract, and first-slice policy SQL fragments. Task 7 intentionally adds no SQL source files, offline contract fixtures, or `databricks.yml`.

Phase 1b remains offline and local for this task; no live Databricks resources are created here.
