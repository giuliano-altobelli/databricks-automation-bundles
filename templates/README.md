# Templates

This directory contains approved scaffolds for projects and bundles in the DAB monorepo foundation.

Phase 1a includes neutral project and bundle metadata templates.

Phase 1b adds the concrete `templates/bundles/abac-access-map/` template. It includes `repoctl.bundle.yaml` metadata, an inert native `databricks.yml`, focused `SPEC.md` guidance, SQL source placeholders, and offline contract-test fixtures for an ABAC access-map bundle.
